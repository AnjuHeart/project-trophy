# TrophyDB (Project-Trophy)

The no-BS game achievement database. Built for completionists who want instant, actionable game insights over player vanity profiles.

## 🎯 The Goal

TrophyDB puts the game data first, instantly detailing:
* **Minimum Playthroughs Required** (e.g., why Dark Souls III requires 3 runs)
* **Completionist Verdicts** (highlighting specific multi-hour grinds or missable thresholds)
* **Streamlined Layout** designed for speed and mobile responsiveness

## 🛠️ The Tech Stack

* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **Language:** TypeScript
